# CRYPTOSEAL

Cryptoseal is a hybrid encryption application developed using Python and Tkinter, designed to ensure secure data transmission. The main idea behind the project is to combine the strengths of two encryption techniques—RSA and AES.
RSA is used for secure key exchange, while AES is used for fast and efficient data encryption. In this system, the data is first encrypted using AES, and then the AES key itself is encrypted using RSA. This approach ensures both high security and good performance.
The application provides a user-friendly interface using Tkinter, where users can easily encrypt and decrypt files or messages. Overall, the project demonstrates how hybrid encryption can be effectively used to protect sensitive data from unauthorized access.
